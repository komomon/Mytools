# ip2domain

通过ssl证书，批量ip反查域名工具。

## Usage

```
usage:
            python3 ip2domain.py -f file.txt


optional arguments:
  -h, --help            show this help message and exit
  -f FILE, --file FILE  the url file
  -o OUTPUT, --output OUTPUT
                        the result file
  -t THREADS, --thread THREADS
                        Threads num, defalut 10
```

文件内容格式：

```
121.40.43.188
121.40.43.188
121.40.43.189:8080
www.cnblogs.com
```
